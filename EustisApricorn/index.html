Project Apricorn

Project Apricorn is a high frequency emitting and recieving prototype inspired the game of pokemon. It is meant to emulate what it would be like to play Pokémon in a theme park scenario. The idea is theme park goers are given Pokéball in which a base station that recieves that Pokéball's signal is then assigned a Pokémon. The player would then be able to interact with the Pokéland system that would be based inside of the said theme park.
